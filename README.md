# Очередной ТГ бот с гороскопвми

Простой полностью асинхронный [Telegram бот](https://t.me/ramilische_horo_bot), отправляющий гороскопы из открытых источников

* Язык - Python
* Взаимодействие с Telegram API через aiogram
* Взаимодествие с БД через ORM SQLAlchemy. Все нужные таблицы создаются автоматически

## Реализовано

* Выбор знака зодиака
* Отправка гороскопа
